Git cheat sheets list essential commands for quick reference. Here's a complete text-based version covering core operations, drawn from standard references like official Git docs.

Configuration text
==================

git config --global user.name "Your Name"
git config --global user.email "email@example.com"
git config --list
git config --global core.editor "vim"

Creating Repos text
===================

git init
git clone <url> [folder]
git remote add origin <url>
git remote -v

Checking Status text
====================

git status
git log --oneline
git log --graph --oneline --all
git diff
git diff --staged
git diff HEAD

Staging and Committing text
===========================

git add <file>
git add .
git rm <file>
git commit -m "Message"
git commit -a -m "Message"
git commit --amend -m "New message"


Branching text
==============

git branch
git branch -d <branch>
git branch -D <branch>
git checkout <branch>
git checkout -b <branch> [start-point]
git switch <branch>
git switch -c <branch>
git merge <branch>


Remote Operations text
======================
git fetch
git pull
git pull --rebase
git push origin <branch>
git push -u origin <branch>
git push --all
git push --tags

Rebasing and Reset text
=======================


git rebase <branch>
git rebase -i HEAD~3
git reset --soft HEAD~1
git reset --mixed HEAD~1
git reset --hard HEAD~1
git reset --hard origin/main

Stashing text
=============

git stash push -m "Message"
git stash list
git stash pop
git stash apply
git stash drop

Tagging text
============


git tag
git tag <tagname>
git tag -a <tagname> -m "Message"
git push origin --tags

Undoing Changes text
====================

git checkout -- <file>
git restore <file>
git clean -fd
git revert <commit>

Viewing History text
====================

git log --author="Author"
git log --since="2025-01-01"
git blame <file>
git show <commit>
git shortlog -sn
